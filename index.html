Welcome to the official resource page for the What's It Worth Desktop Application

Getting Started

System Requirements

  • Windows 10 or later
  • 4GB RAM minimum
  • 1GB free disk space
  • .NET Framework 4.7.2 or higher
  • Internet access

Installation Guide

  1. Download the source files
  2. Run the .exe file
  3. Create an account
  4. Log-In and have fun!

Features

Core Features

  • Stores user information in database
  • Calculate price for each item of jewelry
  • Daily update of price of metal and diamonds

Advanced Features

  • Encryption of user data
  • Generate reports for different items
  • Calculate analytics and growth metrics for items
